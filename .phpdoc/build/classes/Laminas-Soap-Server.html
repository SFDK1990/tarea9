handle()

Server.php : 1037

Handle a request

public handle([DOMDocument|DOMNode|SimpleXMLElement|stdClass|string $request = null ]) : void|string

Instantiates SoapServer object with options set in object, and dispatches its handle() method.

$request may be any of:

  • DOMDocument; if so, then cast to XML
  • DOMNode; if so, then grab owner document and cast to XML
  • SimpleXMLElement; if so, then cast to XML
  • stdClass; if so, calls __toString() and verifies XML
  • string; if so, verifies XML

If no request is passed, pulls request using php:://input (for cross-platform compatibility purposes).

Parameters
$request : DOMDocument|DOMNode|SimpleXMLElement|stdClass|string = null
